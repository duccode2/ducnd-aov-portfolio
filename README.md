# Deploy Notes

This folder is the deploy-ready English-first portfolio bundle.

What is included:

- `index.html` as the landing page
- `resume.html` with embedded CV preview
- `assets/` for styles and the attached PDF resume
- `case-assets/` for project imagery and videos
- `case-studies/` for the 4 selected case pages

Recommended deploy flow:

1. Upload the full folder contents to Netlify, Cloudflare Pages, or GitHub Pages.
2. Keep the directory structure unchanged.
3. Use `index.html` as the site entry.
